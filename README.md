# App Usage Analytics Dashboard

An interactive analytics dashboard built using Python and Streamlit to analyze website/app traffic and user engagement.

## Features
- Daily Active Users (DAU)
- User Retention Rate
- Feature / Page Usage Analysis
- Date Range Filter
- KPI Summary Cards

## Tools Used
- Python  
- Pandas  
- Streamlit  
- Matplotlib  

## How to Run

1. Install libraries:
pip install pandas matplotlib streamlit

2. Run the app:
streamlit run dashboard.py

3. Open in browser:
http://localhost:8501
